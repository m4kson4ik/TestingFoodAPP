Тестовое приложение для просмотра блюд, а также их категорий.
Приложение разработано на языке Kotlin, использовался JetPackCompose
Стек технологий: 
Clean Architecture,
MVVM,
ROOM,
Dagger/Hilt,
Retrofit,
Coil

API - https://www.themealdb.com/
